# Projet de Jeu Stratégique - Intelligence Artificielle

## Description du projet
Ce projet consiste en la conception et la programmation d'un jeu de stratégie au tour par tour. L'objectif principal est de développer une interface permettant à un joueur humain d'affronter une Intelligence Artificielle. Le programme intègre trois niveaux de difficulté pour l'IA, allant d'une stratégie simple à une stratégie optimisée.

Ce travail inclut également une analyse de performance via des tournois automatisés entre les différentes versions de l'IA.
